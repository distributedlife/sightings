List of chordate orders

From Wikipedia, the free encyclopedia

Phylum Chordata

Class Acanthodii - extinct

Class Actinopterygii (ray-finned fish)

Class Amphibia (amphibians)

Class Ascidiacea (ascideans and sessile tunicates)

  • Enterogona
  • Pleurogona
  • Aspiraculata

Class Aves (birds)

  • Subclass Neornithes
    • Superorder Palaeognathae
      • Struthioniformes, Ostrich, emus, kiwis, and allies
      • Tinamiformes, tinamous
    • Infraclass Neognathae
      • Superorder Galloanserae (fowl)
        • Anseriformes, waterfowl
        • Galliformes, fowl
      • Superorder Neoaves
        • Sphenisciformes, penguins
        • Gaviiformes, loons
        • Podicipediformes, grebes
        • Procellariiformes, albatrosses, petrels, and allies
        • Pelecaniformes, pelicans and allies
        • Ciconiiformes, storks and allies
        • Phoenicopteriformes, flamingos
        • Accipitriformes, eagles, hawks and allies (taxonomists have traditionally placed these groups in the Falconiformes)
        • Falconiformes, falcons
        • Turniciformes, button-quail
        • Opisthocomiformes, Hoatzin (this enigmatic bird was traditionally treated as a family within either the Galliformes or Cuculiformes)
        • Gruiformes, cranes and allies
        • Charadriiformes, plovers and allies
        • Pterocliformes, sandgrouse (this enigmatic group was traditionally treated as a family in any of three different orders: Charadriiformes, Ciconiiformes, and Columbiformes)
        • Columbiformes, doves and pigeons
        • Psittaciformes, parrots and allies
        • Cuculiformes, cuckoos
        • Strigiformes, owls
        • Caprimulgiformes, nightjars and allies
        • Apodiformes, swifts
        • Coliiformes, mousebirds
        • Trogoniformes, trogons
        • Coraciiformes, kingfishers
        • Galbuliformes, jacamars and puffbirds (these groups were traditionally treated as families in the order Piciformes)
        • Piciformes, woodpeckers and allies
        • Passeriformes, passerines

Class Chondrichthyes (cartilaginous fish)

  • Subclass Elasmobranchii
    • Superorder Batoidea
      • Rajiformes: rays and skates
      • Pristiformes: sawfishes
      • Torpediniformes: electric rays
    • Superorder Selachimorpha (sharks)
      • Heterodontiformes: bullhead sharks
      • Orectolobiformes: carpet sharks
      • Carcharhiniformes: ground sharks
      • Lamniformes: mackerel sharks
      • Hexanchiformes: frilled and cow sharks
      • Squaliformes: dogfish sharks
      • Squatiniformes: angel sharks
      • Pristiophoriformes: saw sharks
  • Subclass Holocephali
    • Chimaeriformes: chimaeras

Class Larvacea (solitary, free-swimming tunicates)

Class Leptocardii (lancelets)

  • Amphioxiformes

Class Mammalia (mammals)

  • Subclass Prototheria
    • Monotremata: monotremes
  • Subclass Theria
    • Infraclass Marsupialia
      • Didelphimorphia: opossums
      • Paucituberculata: rat opossums
      • Microbiotheria: Monito del Monte
      • Dasyuromorphia: marsupial carnivores
      • Peramelemorphia: marsupial omnivores
      • Notoryctemorphia: marsupial moles
      • Diprotodontia: marsupial herbivores; kangaroos, wallabies, possums and allies
    • Infraclass Eutheria
      • Magnorder Atlantogenata
        • Superorder Afrotheria
          • Grandorder Afrosoricida
            • Afrosoricida: tenrecs and golden moles
            • Macroscelidea: elephant shrews
            • Tubulidentata: Aardvark
          • Grandorder Paenungulata
            • Hyracoidea: hyraxes
            • Mirorder Tethytheria
              • Proboscidea: elephants
              • Sirenia: manatees and dugongs
        • Superorder Xenarthra
          • Cingulata: armadillos
          • Pilosa: sloths and anteaters
      • Magnorder Boreoeutheria
        • Superorder Laurasiatheria
          • Grandorder Eulipotyphla
            • Erinaceomorpha: hedgehogs
            • Soricomorpha: shrews, moles
          • Grandorder Ferungulata
            • Cetartiodactyla: cetaceans and even-toed ungulates
            • Clade Pegasoferae
              • Chiroptera: bats
              • Mirorder Zooamata
                • Clade Ferae
                  • Pholidota: pangolins
                  • Carnivora: carnivores; cats, dogs, bears, seals, sea lions and others
                • Perissodactyla: odd-toed ungulates; horses, rhinos, tapirs
          • Creodonta hyaenodontidae hyeanodon, dissopsalis, sarkastostodon, and megistotherium.
        • Superorder Euarchontoglires
          • Grandorder Euarchonta
            • Mirorder Sundatheria
              • Dermoptera: colugos
              • Scandentia: treeshrews
            • Mirorder Primatomorpha
              • Primates: lemurs, monkeys, apes and allies
          • Grandorder Glires
            • Rodentia: rodents
            • Lagomorpha: rabbits, hares and pikas

Class Myxini (hagfishes)

  • Myxiniformes

Class Petromyzontida (lampreys and their extinct kin)

  • Petromyzontiformes

Class Placodermi - extinct

  • Acanthothoraci
  • Arthrodira
  • Antiarchi
  • Brindabellaspida
  • Petalichthyida
  • Phyllolepida
  • Ptyctodontida
  • Rhenanida
  • Pseudopetalichthyida (The placement of this order is debated.)
  • Stensioellida (The placement of this monotypic order is debated.)

Class Reptilia (reptiles)

  • Subclass Diapsida
    • Infraclass Archosauromorpha
      • Division Archosauria
        • Subdivision Crurotarsi
          • Infradivision Pseudosuchia
            • Magnorder Suchia
              • Superorder Crocodylomorpha
                • Crocodilia (Crocodilians)
    • Infraclass Lepidosauromorpha
      • Superorder Lepidosauria
        • Rhynchocephalia (Tuataras)
        • Squamata (Lizards, Snakes)
  • Subclass Anapsida
    • Testudines (Turtles and their kin)

Class Sarcopterygii (fleshy-finned fishes)

  • Subclass Actinistia (coelacanths)
    • Coelacanthiformes, the coelacanth
  • Subclass Dipnoi (lungfish)
    • Ceratodontiformes
    • Lepidosireniformes

Class Sorberacea (benthic tunicates)

Class Thaliacea (pelagic tunicates)

  • Doliolida
  • Pyrosomida
  • Salpida, salps
